Core feature · Multiverse navigation

Every turn is a save point.
Every save point is a branch.

The dojo silently snapshots state at the start of every turn. Rewind to turn 4, replay your hand differently, and the original line stays alive next to it. Both lines live in one tree you can zoom, pan, and jump through.

Rewind, instantly

Every node is a full game state snapshot. Jump to turn 2 of game 1 from turn 9 of game 4 with no replay step.

Branch without losing

When you jump back, the line you abandoned auto-saves to a side branch. Compare both outcomes side-by-side.

Reshuffle & redraw

Don't like the hand? Reshuffle from any node to see what a different opening draw does to the same matchup.

What's in the box

A real sandbox, not a referee.

The dojo never blocks you. Play an illegal move if it surfaces a line worth checking. The point is theorycraft, not enforcement.

USP

Multiverse tree

Zoomable, draggable, keyboard-navigable canvas of every branch you've explored. Click a node, you're there.

Save

Auto-save every turn

Toggle on and the dojo bookmarks the start of every turn for you. Name them, comment them in Markdown, color-code them.

Manual

Hot-swap any card

Drag cards between hand, field, ink, and discard. Substitute a card mid-line. Add a card that isn't in your deck to test it.

Variance

Reshuffle the unknown

Lock the board state, reshuffle the deck, redraw. Test the same position against a fresh top of deck without losing the line.

Engine

Live win-prob meters

BCR, LVI, RDS, CTL update in real time. Tug-of-war bars show the matchup shifting turn by turn.

Library

Decklist database

Pull decks from the shared library or paste a list. Fuzzy search across thousands of community lists.

Win-probability engine

Numbers on the board, not vibes in your head.

Every card in every zone is run through the Unified Win Probability library. Four metrics — BCR, LVI, RDS, CTL — update on every action so you can see which branch is actually winning, not which one feels stronger.

BCR
Board Control Rating

Who actually owns the table right now.

LVI
Lore Velocity Index

How fast you're trending toward 20.

RDS
Resource Development Score

How many resources can you develop (draw, ramp, etc)

CTL
Card Threat Level

BCR + LVI + RDS combined as a single metric. How much of a threat is an individual card.
